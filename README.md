# Nort

**Plan the month. See the years.**

A bilingual (EN + PT, UK + BR) personal-finance planner. Allocate your salary across pillars, track goals, project your wealth across years. Not a transaction tracker — a forecast and allocation tool.

## Stack
- React 19 + Vite
- PWA (manifest + service-worker-less, all client-side)
- localStorage for persistence

## Scripts
- `npm run dev` — dev server
- `npm run build` — production build
- `npm run lint` — eslint
- `npm run preview` — preview the build
